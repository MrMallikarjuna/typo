Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed. Key elements include:

  • Typefaces vs Fonts: A typeface is a design style (like Helvetica), while a font is a specific implementation (like Helvetica Bold 12pt).
  • Kerning: The space between individual letter pairs.
  • Leading: The vertical space between lines of text.
  • Tracking: The uniform spacing between all characters in a word.

Selecting the appropriate font depends on several factors:

  1. Purpose: Formal documents need different fonts than creative projects.
  2. Audience: Consider readability for your target demographic.
  3. Medium: Screen vs print fonts have different requirements.
  4. Brand Identity: Fonts should align with your brand personality.

Effective font combinations create visual harmony:

  • Pair a serif with a sans-serif for contrast
  • Limit to 2-3 fonts per project
  • Ensure sufficient contrast between heading and body fonts
  • Consider using different weights of the same font family
  • Test readability at various sizes

Adobe Creative Cloud

Adobe Creative Cloud is a collection of 20+ desktop and mobile apps and services for photography, design, video, web, UX, and more. Popular apps include:

  • Photoshop - Image editing and compositing
  • Illustrator - Vector graphics and illustration
  • InDesign - Page layout and publishing
  • Premiere Pro - Video editing
  • After Effects - Motion graphics and visual effects

Subscription plans start at $9.99/month for individual apps.

Figma

Figma is a collaborative web application for interface design, with additional offline features enabled by desktop applications. Key features include:

  • Real-time collaboration with team members
  • Vector networks for flexible design
  • Powerful prototyping tools
  • Design systems with shared components
  • Plugins for extended functionality

Figma offers free plans for individuals and paid plans for teams.

Google Fonts

Google Fonts is a library of 1,000+ free licensed font families and APIs for convenient use via CSS and Android. Benefits include:

  • All fonts are open source and free for commercial use
  • Optimized for fast loading on websites
  • Easy integration with just a few lines of code
  • Wide selection of font styles and languages
  • No registration or account required

Popular Google Fonts include Roboto, Open Sans, Lato, Montserrat, and Poppins.
